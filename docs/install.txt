	1) Debian
				
> sudo apt-get install imagemagick
> sudo apt-get install libmagickwand-dev
> sudo gem install rmagick

	2) Windows

> set CPATH=%IMAGEMAGICK_HOME%\include
> set LIBRARY_PATH=%IMAGEMAGICK_HOME%\lib
> gem install rmagick
